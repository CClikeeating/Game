# Baiou 小程序提审材料草案

更新日期：2026-06-18

本文档用于 PM 准备微信小程序提审材料。以下内容是产品侧草案，正式发布前建议由主体负责人或法务确认。

## 一、服务定位

建议小程序名称/简介避免写成心理咨询、婚恋介绍、交友撮合或医疗健康服务。

推荐表达：

```text
Baiou 是一个聊天回复辅助工具。用户可以输入对方上一句话，或上传聊天截图，系统生成可参考的回复建议和简短分析，帮助用户更自然地延续日常沟通。
```

不建议表达：

```text
情感咨询、心理咨询、恋爱挽回、婚恋撮合、脱单服务、私教咨询、医疗/心理诊断。
```

原因：当前产品提供的是 AI 生成的聊天回复建议，不提供真人咨询、婚恋撮合、心理健康诊断或治疗。

## 二、小程序类目建议

类目最终以微信公众平台后台实时可选项为准。

首选建议：

```text
工具 / 效率
```

选择理由：产品核心是“文本/截图输入 -> 生成聊天回复建议”的效率工具，第一版不提供社交关系撮合、不提供真人咨询、不销售课程。

备选建议：

```text
工具 / 图片/音频/视频
```

选择理由：如果审核更关注“上传聊天截图并识别内容”，可以按图片处理工具解释。但产品核心不是图片编辑，所以优先级低于“效率”。

谨慎备选：

```text
教育 / 在线教育
```

仅当后续把产品改成课程、话术学习、案例学习内容，并弱化“帮你生成当下回复”时再考虑。

不建议：

```text
社交、婚恋、心理咨询、医疗健康、法律服务
```

这些类目通常更容易触发资质、内容安全或服务边界审核问题；当前产品也不属于这些服务。

## 三、用户隐私保护指引填写建议

建议在微信公众平台的小程序用户隐私保护指引中按实际能力填写：

| 信息类型 | 是否收集 | 使用目的 | 场景 |
| --- | --- | --- | --- |
| 微信用户标识 openid | 是 | 登录、生成本地账号、识别每日额度 | 用户进入小程序并登录 |
| 用户主动填写的昵称、头像 | 是 | 账号展示、后台识别和用户服务 | 用户在“我的”页完善资料 |
| 用户输入文本 | 是 | 生成聊天回复、保存会话历史、问题排查 | 用户提交文字输入或问题 |
| 用户上传图片/截图 | 是 | 理解聊天截图、生成回复建议、人工审核反馈 | 用户选择“截图回复”并上传图片 |
| 反馈内容 | 是 | 质量改进、人工审核、问题定位 | 用户点击反馈并填写备注 |
| 设备网络信息/IP 日志 | 是 | 安全风控、额度限制、排查异常请求 | 用户登录、上传、生成回复 |
| 兑换码 | 是 | 兑换更多额度、校验兑换状态 | 用户在“我的”页输入兑换码 |

建议补充说明：

```text

```

可放在小程序上传区、隐私政策和提审说明中：

```text
截图仅用于本次聊天内容理解、生成回复建议、服务质量分析和必要的问题排查。系统不会将截图用于公开展示、广告投放或与本服务无关的用途。截图和运行明细默认保留 30 天，到期后自动清理。后台管理员可在授权范围内下载包含截图的审核包，用于人工复核用户反馈和模型效果。
```

## 六、用户协议草案

```text
《Baiou 用户服务协议》

欢迎使用 Baiou。本服务由运营方提供聊天回复辅助能力，帮助用户基于输入文字或聊天截图生成可参考的回复建议。

1. 服务内容
Baiou 提供文字输入、截图回复、回复生成、简短分析、反馈、额度和兑换码等功能。生成内容仅供参考，用户应自行判断是否采纳。

2. 用户责任
用户应保证上传或输入的内容来源合法，不得上传违法、侵权、骚扰、暴力、色情、诈骗、侵犯他人隐私或其他不当内容。用户不得利用本服务从事违法违规活动。

3. 内容边界
Baiou 不提供心理咨询、医疗诊断、法律意见、婚恋撮合或真人情感咨询服务。生成回复不构成专业建议。

4. 额度规则
第一版免费使用。默认每人每日 10 点额度。日常接话消耗 1 点，暧昧推荐消耗 2 点。更多额度可联系作者 QQ：1179123330，或通过兑换码获取。具体额度规则以后端展示为准。

5. 数据处理
为提供服务，Baiou 会处理用户输入文本、上传截图、反馈、登录标识、额度和必要日志。具体规则见《Baiou 隐私政策》。

6. 服务变更
运营方可根据产品迭代调整功能、额度、模型和使用规则，并在合理范围内通过公告或页面提示告知用户。

7. 联系方式
如有问题，请联系 QQ：1179123330。
```

## 七、隐私政策草案

```text
《Baiou 隐私政策》

我们重视你的隐私。本政策说明 Baiou 如何收集、使用、保存和保护你的信息。

1. 我们收集的信息
为提供服务，我们可能收集：
- 微信登录标识：用于登录、账号识别和额度统计。
- 用户输入内容：包括对方上一句话、用户问题、补充背景、会话背景。
- 上传截图：用于理解聊天内容并生成回复建议。
- 生成结果和反馈：用于展示历史结果、质量分析和问题排查。
- 额度和兑换记录：用于统计每日免费额度和兑换码使用情况。
- 网络与日志信息：用于安全风控、排查异常、统计服务运行状态。

2. 我们如何使用信息
我们仅将上述信息用于：
- 登录和识别用户；
- 生成聊天回复建议和分析；
- 保存会话上下文，提高连续使用体验；
- 统计和扣减额度；
- 处理用户反馈、人工审核和服务质量改进；
- 排查故障、防止滥用和保障系统安全。

3. 截图和敏感信息
聊天截图可能包含头像、昵称、聊天内容等个人信息。请勿上传身份证、银行卡、住址、手机号、医疗记录等敏感信息。上传前请确认你有权使用该截图。

4. 保存期限
聊天截图和模型运行明细默认保留 30 天，到期后自动清理。用户会话、反馈、额度和兑换记录会在提供服务所需期间保存。你可以联系运营方申请删除。

5. 信息共享
我们不会出售你的个人信息。为生成回复，用户输入和截图可能会经过我们部署的后端服务及所接入的模型服务处理。后台授权人员可查看反馈和审核材料，用于人工复核和服务改进。

6. 信息安全
我们会采取访问控制、后台密码登录、HTTPS、最小权限和定期清理等措施保护信息安全。但互联网服务无法保证绝对安全，请避免上传与本服务无关的敏感信息。

7. 你的权利
你可以联系运营方申请访问、更正或删除相关信息，也可以停止使用本服务。

8. 联系方式
如对本政策有疑问，请联系 QQ：1179123330。
```
